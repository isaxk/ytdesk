# YT Desk
### [ytdesk.isaxk.com](https://ytdesk.isaxk.com)
Youtube & Youtube Music Supercharged

## Features
- Discord Rich Presence
- Accent Colour Theming
- Ad blocking
- Force Cinema Mode

## Project Setup

**Note**: This project uses pnpm.

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
